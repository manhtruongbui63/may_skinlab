# BKS Design System — vendor source

Version: `0.1.18`

This folder is generated by `pnpm run build:vendor` in the design-system repo.
Do not edit here; changes are overwritten on publish.

## Consumer install

```bash
pnpm add @bks/ds-system-sdk@0.1.18
pnpm exec ds-vendor --target bks/ds-system-sdk
```

## Consumer usage

```ts
import { Button } from "@bks/ds-system-sdk"
import "@bks/ds-system-sdk/styles.css"
```

Add to `tsconfig.json`:

```json
{
  "compilerOptions": {
    "paths": {
      "@bks/ds-system-sdk": ["./bks/ds-system-sdk/index.ts"],
      "@bks/ds-system-sdk/*": ["./bks/ds-system-sdk/*"]
    }
  }
}
```

Import styles in app entry:

```ts
import "@bks/ds-system-sdk/styles.css"
```
